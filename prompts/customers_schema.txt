Table: customers
Columns:
- client_id (TEXT PRIMARY KEY): Unique identifier for the client (e.g., 'C001', 'C002')
- client_name (TEXT NOT NULL): Name of the client/business
- client_city (TEXT NOT NULL): City location (e.g., 'San Diego')
- client_state (TEXT NOT NULL): State location (e.g., 'CA')
- postal_code (TEXT NOT NULL): Postal/ZIP code
- top3_brewery_types (TEXT NOT NULL): JSON array of top 3 preferred brewery types (e.g., '["micro", "brewpub", "regional"]')
- top5_beers_recently (TEXT NOT NULL): JSON array of top 5 most ordered beers
- top3_breweries_recently (TEXT NOT NULL): JSON array of top 3 brewery names from recent purchase history
